Vue + Express boilerplate for development
=========================================

## Keywords

- Vue
- Express
- Nodemon
- Webpack
- Npm


## Structure

```
.
├── LICENSE
├── README.md
├── index.js
├── nodemon.json
├── package.json
├── src
│   ├── client
│   │   ├── App.vue
│   │   ├── components
│   │   │   └── Hello.vue
│   │   └── index.js
│   └── server
│       ├── index.js
│       ├── public
│       │   └── favicon.ico
│       ├── router.js
│       └── views
│           ├── error.jade
│           └── index.jade
└── webpack.config.js
```

## Usage

1. Install dependencies

   `npm install`

2. Run the application

   `npm run dev`
